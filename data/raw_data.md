# 🎶 What Does This Dataset Contain?
Each row in the dataset represents a music track, enriched with multiple layers of information:

🔹 1. Track & Album Metadata
- Track ID, name, and artist
- Album details and release date
- Playlist name and playlist ID
🔹 2. Genre Information
- Playlist genre
- Playlist sub-genre
Covers multiple genres such as pop, rock, rap, EDM, R&B, latin, etc.
🔹 3. Spotify Audio Features
Extracted numerical features commonly used in music analysis:
- Danceability
- Energy
- Key & Mode
- Loudness
- Speechiness
- Acousticness
- Instrumentalness
- Liveness
- Valence
- Tempo
- Track duration (in milliseconds)
These features are highly useful for ML-based music classification and recommendation systems.

🧠 Mental Health Labels
The dataset includes a Mental_Health_Label column that categorizes tracks into mental/emotional states such as:

- Normal / Unclassified
- Anxiety
- Bipolar (Mania)
This makes the dataset suitable for:

Mental health classification
Emotion-aware music recommendation
Research in music therapy and affective computing
